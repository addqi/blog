## 概述

小地图的实现原理很简单：**添加一个额外的摄像机，设置为正交投影（垂直俯视），然后将摄像机渲染的画面输出到 UI 的 Texture 上显示**。

## 实现步骤

### 1. 创建摄像机并设置

- 在场景中创建一个新的摄像机
- 添加跟随人物的脚本（让摄像机始终跟随玩家）
- 将摄像机设置为**正交投影（Orthographic）**
- 调整摄像机角度，使其垂直向下俯视场景

### 2. 创建渲染纹理

- 在 Unity 中创建一个**渲染纹理（Render Texture）**文件
- 将渲染纹理拖入场景，并放置在你想要显示小地图的 UI 位置

{% asset_img 纹理.png 渲染纹理设置 %}

### 3. 关联摄像机与纹理

- 将摄像机的**输出目标（Target Texture）**设置为刚才创建的渲染纹理
- 这样摄像机渲染的画面就会直接输出到 UI 的 Texture 上

{% asset_img 摄像机设计.png 摄像机设计 %}

## 完成

完成以上步骤后，小地图就可以正常显示了！玩家移动时，小地图会实时更新显示场景的俯视图。

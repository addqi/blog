小地图的实现

发表于 2025-11-12 更新于 2026-01-19
本文字数： 348 阅读时长 ≈ 1 分钟

概述

小地图的实现原理很简单：添加一个额外的摄像机，设置为正交投影（垂直俯视），然后将摄像机渲染的画面输出到 UI 的 Texture 上显示。

实现效果

实现步骤

1. 创建摄像机并设置

  • 在场景中创建一个新的摄像机
  • 添加跟随人物的脚本（让摄像机始终跟随玩家）
  • 将摄像机设置为正交投影（Orthographic）
  • 调整摄像机角度，使其垂直向下俯视场景

2. 创建渲染纹理

  • 在 Unity 中创建一个渲染纹理（Render Texture）文件
  • 将渲染纹理拖入场景，并放置在你想要显示小地图的 UI 位置

3. 关联摄像机与纹理

  • 将摄像机的输出目标（Target Texture）设置为刚才创建的渲染纹理
  • 这样摄像机渲染的画面就会直接输出到 UI 的 Texture 上

完成

完成以上步骤后，小地图就可以正常显示了！玩家移动时，小地图会实时更新显示场景的俯视图。
